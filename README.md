Frontend feestdagen opdracht Raymond

Met deze gemaakte code door mij kun je op de urenregistratie website de komende feestdagen bekijken in Nederland.

- Download allebei de files.
- Ga naar je bestanden toe en zoek het bestand "index.html".
- Dubbelklik op het bestand.
- Nu is het bestand geopend en kan je zien wat de aankomende feestdagen zijn.
